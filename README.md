# [make-your-game]

## Description
This project is a fork from [make-your-game].  
The goal of this fork is to **learn and experiment** with high-performance JS game development.

Features include:
- Game runs at **60 FPS** consistently with no frame drops
- Smooth keyboard controls: hold key to continue action
- Pause menu with Continue and Restart
- Scoreboard with Timer, Score, and Lives
- Minimal layer usage for optimal rendering
- Implemented using **plain JavaScript/DOM and HTML only**

Game genre inspired by: [Bomberman / Space Invaders / Tetris / etc.]
